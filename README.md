**Prerequisites**
1. Java JDK 17
2. Apache Maven 3.8+
3. Google Chrome (latest stable version)
4. Intellij IDEA

**Environment**
1. Programming Language: Java
2. Test Framework: TestNG 7.10.2
3. Automation Tool: Selenium WebDriver 4.20.0
